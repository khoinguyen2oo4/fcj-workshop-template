* Build a hybrid DNS architecture utilizing Amazon Route 53 Resolver.

* **Networking & DNS (Route 53):** Successfully established a hybrid DNS architecture, enabling seamless domain name resolution and query forwarding between simulated on-premise systems and AWS.
