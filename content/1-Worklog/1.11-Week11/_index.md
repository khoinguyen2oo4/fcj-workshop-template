### Week 11 Objectives:
* Apply the Infrastructure as Code (IaC) mindset to automate the deployment and management of AWS resources.
* Build and manage AWS architectures using popular programming languages through the AWS Cloud Development Kit (CDK).
* Master the core principles of NoSQL data modeling, advanced features, and how to integrate Generative AI with Amazon DynamoDB.

### Week 11 Achievements:
* **Infrastructure Automation (IaC):** Proficient in using CloudFormation and Cloud9 IDE to define and provision a complete set of AWS resources, minimizing the risks associated with manual operations.
* **Expanding Cloud Programming Capabilities (AWS CDK):** Mastered the process of designing infrastructure using familiar programming languages, making it easy to leverage loops, variables, and manage architecture versions.
* **Mastering NoSQL Databases:** Deeply understood the tradeoffs when designing Data models on DynamoDB to achieve single-digit millisecond performance. Grasped the process of integrating real-time events (Streams/Lambda) and techniques for combining traditional data with Generative AI models for natural language search.
