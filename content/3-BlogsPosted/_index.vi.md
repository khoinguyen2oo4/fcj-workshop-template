###  [Blog 1 - ĐƯA AI VÀO XÂY DỰNG ỨNG DỤNG NHẬN DIỆN CALO & PROTEIN CÙNG AWS SERVERLESS](3.1-Blog1/)
Blog này giới thiệu kiến trúc thực chiến xây dựng "AI Diet Tracker" – trợ lý cá nhân tự động phân tích chế độ dinh dưỡng bằng cách kết hợp AWS Serverless và trí tuệ nhân tạo (Amazon Rekognition). Giải pháp giúp chuyển hóa việc tính toán macro thủ công thành thao tác chụp ảnh đơn giản, với dữ liệu được xử lý qua AWS Lambda và lưu trữ siêu tốc trên Amazon DynamoDB, đảm bảo linh hoạt mở rộng và tối ưu chi phí duy trì ở mức tối thiểu.

###  [Blog 2 - TỐI ƯU HÓA HIỆU NĂNG ỨNG DỤNG SPRING BOOT SERVERLESS TRÊN AWS LAMBDA VỚI SNAPSTART](3.2-Blog2/)
Blog này chia sẻ giải pháp khắc phục triệt để rào cản "Cold Start" cho các hệ thống Java hiện đại khi đưa lên môi trường Serverless. Bằng cách ứng dụng AWS Lambda tích hợp tính năng SnapStart, quy trình khởi động Application Context nặng nề của Spring Boot được chuyển hóa thành cơ chế khôi phục trạng thái tức thì. Hệ thống kết hợp cùng Amazon RDS và Secrets Manager giúp doanh nghiệp hiện đại hóa Microservices an toàn, ổn định và tối ưu chi phí vận hành.

###  [Blog 3 - TỐI ƯU HIỆU NĂNG VÀ GIẢM TẢI DB: KIẾN TRÚC CACHE-ASIDE VỚI SPRING BOOT VÀ ELASTICACHE](3.3-Blog3/)
Blog này chia sẻ giải pháp kiến trúc thực chiến giúp tăng tốc độ phản hồi API và giải quyết triệt để bài toán nghẽn cổ chai cho cơ sở dữ liệu quan hệ (Amazon RDS). Bằng cách áp dụng chiến lược Cache-Aside (Lazy Loading) với Amazon ElastiCache (Redis) kết hợp cùng Spring Boot trên môi trường Serverless, hệ thống đạt được độ trễ siêu thấp và tối ưu chi phí vận hành hiệu quả.
