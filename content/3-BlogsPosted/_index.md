###  [Blog 1 - INTEGRATING AI TO BUILD A CALORIE & PROTEIN TRACKING APP WITH AWS SERVERLESS](3.1-Blog1/)
This blog introduces a practical architecture for building the "AI Diet Tracker" – an automated personal assistant for nutritional analysis, combining AWS Serverless and artificial intelligence (Amazon Rekognition). This solution transforms manual macro calculation into a simple photo capture. Data is processed via AWS Lambda and stored at lightning speed on Amazon DynamoDB, ensuring flexible scalability and optimizing maintenance costs to a minimum.

###  [Blog 2 - OPTIMIZING SPRING BOOT SERVERLESS PERFORMANCE ON AWS LAMBDA WITH SNAPSTART](3.2-Blog2/)
This blog shares a solution to completely overcome the "Cold Start" barrier for modern Java systems when migrating to a Serverless environment. By utilizing AWS Lambda integrated with the SnapStart feature, the heavy Application Context startup process of Spring Boot is transformed into an instant state resume mechanism. The system, combined with Amazon RDS and Secrets Manager, helps businesses modernize Microservices securely, stably, and with optimized operational costs.

###  [Blog 3 - PERFORMANCE OPTIMIZATION AND DB OFFLOADING: CACHE-ASIDE ARCHITECTURE WITH SPRING BOOT AND ELASTICACHE](3.3-Blog3/)
This blog shares a practical architectural solution that accelerates API response times and completely resolves the bottleneck issue for relational databases (Amazon RDS). By applying the Cache-Aside (Lazy Loading) strategy with Amazon ElastiCache (Redis) combined with Spring Boot in a Serverless environment, the system achieves ultra-low latency and effectively optimizes operational costs.
